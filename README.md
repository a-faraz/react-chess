# React-Chess

---

An online chess and chat application built with React & NodeJS.

Demo: http://react-chesss.herokuapp.com/#/

### Tech Stack

- React
- NodeJS
- jQuery
- Express
- Socket.io
- PostgreSQL
- KNEX

### Getting Started

`$ npm install ` <br />
`$ npm start `

Then visit [localhost:4000](http://localhost:4000/)

### Libraries

- Chessboard.js
